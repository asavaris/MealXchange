About MealXchange



How to MealXchange - Member | How to MealXchange - Admin | Testimonials

At Princeton, the current pen-and-paper system for monitoring meal exchanges between eating clubs is inconvenient for everyone involved. Members don't know how many meal exchanges they have outstanding with their friends and club administrators spend hours manually tabulating meal exchanges at the end of each month. We're proposing a system where instead of using paper slips, students simply register their meal exchanges and guest meals online on our webapp. Students can receive updates on how many meal exchanges they have outstanding as well as how many guest meals they have left in the month. Eating clubs can edit memberships and club preferences, as well as view all exchanges online or by downloading a .csv file with all the computations done for them. Fewer lost slips of paper, fewer outstanding meal exchanges, fewer hours of manual tabulation. Everyone's a winner!



How to MealXchange - For Club Members

First is the login page for eating clubs.


Upon successful authentication of the club account, the user is brought to the home page, where the user has the option to choose to do a meal exchange or host a guest


From here, all that is left is to enter the necessary NetIDs for the exchange/guest and confirm!




How to MealXchange - For Club Administrators

Using the menu button in the top right corner of the home page, club administrators can effectively change settings and view outstanding Xchanges.



In the Club Preferences page, clubs can change the time periods for each meal, and set the montly number of allotted guests



In the Edit Membership page, clubs can add and delete members to the current roster of club members for authentication



The View Exchanges page allows clubs to see the outstanding exchanges, download a .csv file with exchange information, and send update emails to club members.





Congratulations! You've just learned to MealXchange!
